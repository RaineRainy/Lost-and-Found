> Lost your keys? Your AirPods? Or maybe your lost soul? Let's track them down using code and ancient mysticism. 

**Lost & Found** is a lightweight, cyberpunk/pixel-art styled web tool that calculates the potential location of your missing items. It bridges the gap between modern UI design and traditional Chinese divination (**Tian Gan Di Zhi** / Heavenly Stems and Earthly Branches).

## ✨ Features

* **Mystical Algorithm:** Uses the exact date and time of loss to calculate the astrological coordinates, returning a suggested direction, distance, and association clues.
* **Vibe-Driven UI:** Features a stark, black-and-white brutalist/pixel-art design with smooth card transitions.
* **Bilingual Support (EN / ZH):** Instantly toggle between English and Simplified Chinese with a single click.
* **Fully Responsive:** Looks and feels like a native app on mobile devices, but works perfectly on desktop.
* **Zero Dependencies (Mostly):** Built purely with vanilla HTML, CSS, and JavaScript. 

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **External Library:** [lunar-javascript](https://github.com/6tail/lunar-javascript) (Used for calculating the Chinese lunar calendar and astrological data).
* **Fonts:** System fonts + Google Fonts ('Silkscreen' for that retro pixel loading vibe).
